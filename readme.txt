Welcome to Zork Game.

First, you must make sure that you are in the correct directory which are "../../OOC-Zork"
Secondly, type 'mvn package' to build the project. If you build correctly, you will get the folder 'target'
Lastly, go to tar get folder and type 'java -jar Zork-1.0-SNAPSHOT.jar' to compile the project.
